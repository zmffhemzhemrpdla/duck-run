# 🦆 꽥꽥 런 (Quack Run)

분홍 앞치마에 후라이팬을 든 문어 셰프에게 쫓기는 노란 오리! 평화로운 강물 위 3갈래 길에서 돌은 점프로, 나무다리는 잠수로 피하며 별젤리를 모아요.

**▶ 플레이: https://zmffhemzhemrpdla.github.io/duck-run/**

- 컴퓨터: ←→(A/D) 길 이동 · ↑/Space 점프 · ↓/S 잠수
- 핸드폰: 좌우 스와이프 이동 · 위로 점프 · 아래로 잠수
- ⭐ 10개마다 "오이시이네~" · 50개마다 10초 비행(구름젤리)
- 🕊️ 피스 모드: 100개마다 오리 색 변신 · ⛈️ 코와이 모드: 100개마다 폭풍이 거세지며 가속!
